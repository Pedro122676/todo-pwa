# 📋 Minhas Tarefas - PWA

Uma Progressive Web App (PWA) moderna de lista de tarefas, desenvolvida com **HTML, CSS e JavaScript puro**, que funciona **totalmente offline** após o primeiro acesso.

![Badge](https://img.shields.io/badge/Status-Concluído-brightgreen)
![Badge](https://img.shields.io/badge/Tecnologia-PWA-blue)
![Badge](https://img.shields.io/badge/Licença-MIT-yellow)

---

## ✨ Funcionalidades

- ✅ Adicionar, marcar e excluir tarefas
- ✅ Persistência de dados mesmo offline (`localStorage`)
- ✅ Funcionamento completo offline
- ✅ Instalável no celular e desktop (PWA)
- ✅ Design responsivo e moderno
- ✅ Service Worker com cache de recursos
- ✅ Status de conexão em tempo real

---


## 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS3** (Flexbox + Gradientes)
- **JavaScript Vanilla** (ES6+)
- **Service Worker**
- **Web App Manifest**
- **LocalStorage**

---

## 📱 Recursos de PWA

- **Offline First**: A aplicação funciona sem internet após o primeiro carregamento
- **Instalável**: Pode ser instalada como um aplicativo nativo
- **Lifecycle do Service Worker**: Instalação, ativação e cache strategy
- **Cache de recursos**: Arquivos estáticos são armazenados localmente
